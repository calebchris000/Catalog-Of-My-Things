# 📖 [Catalog of my things] <a name="about-project"></a>

**[Catalog of my things]** is a project at <a href='https://www.microverse.org/'>Microverse</a>. It's an app that  keeps records of books, games, and music album.

## 💻 Getting Started <a name="getting-started"></a>

### Preriquisites

Ruby version >= 2.7.0

### Setup

Clone this repository to your desired folder:
sh
  cd my-folder
 git clone https://github.com/calebchris000/Catalog-Of-My-Things.git

 ### Install:
 Run: bundle install

 ### Start:
 Run: ruby main.rb

 ### Tests:
 Run: rspec
